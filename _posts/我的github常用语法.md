# git

>版本控制(VCS)：采用简单的**数据库**来记录文件的**历次更新差异**

例如 word 文件在编辑时会覆盖之前的版本，这样会“写错文件”、“覆盖文件”等意外失误。

常见版本控制软件：Subversion（简称SVN）、Cvs、Git 等。
